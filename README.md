# Formex Landing Page

This is a responsive corporate B2B website for aluminum profile systems, built with pure HTML, Tailwind CSS (via CDN), and Vanilla JavaScript.

## How to Deploy

No build process is required.

### Local Development
1.  Serve the folder using a static file server.
2.  Open `index.html`.

### GitHub Pages
1.  Upload `index.html` to your GitHub repository.
2.  Go to Settings -> Pages.
3.  Select the `main` branch as the source.

## Structure
*   **index.html**: Contains the complete source code: structure, styles (custom + Tailwind), SVG icons, and JavaScript logic.
